# Rauxa Services Engineering Code Challenge

## Hosted Application

http://rauxa-swatches.s3-website-us-east-1.amazonaws.com/

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Tech Stack

This application was prodly made with [NestJS](http://nestjs.com/).
